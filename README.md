# file_randomizer
CLI to randomize file names in a directory


## Prerequisites:
- Install [`Rust`](https://www.rust-lang.org/tools/install)
   ```
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

## Execute CLI:
```
   cargo -- <PATH> [--dry-run]
```

## CLI Help Command:
```
   cargo -- --help
```
